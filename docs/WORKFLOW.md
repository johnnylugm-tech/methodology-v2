## 錯誤處理流程

```
執行任務
    ↓
成功？ → 是 → 下一個任務
    ↓ 否
錯誤分類（L1-L6）
    ↓
L1-L2 → 自動重試 + Fallback
    ↓
L3-L4 → 記錄 + 降級
    ↓
L5-L6 → Human Intervention
```

## Quality Gate 閾值

| 維度 | 閾值 | 權重 |
|------|------|------|
| 正確性 | >= 80% | 30% |
| 安全性 | >= 100% | 25% |
| 可維護性 | >= 70% | 20% |
| 效能 | >= 80% | 15% |
| 覆蓋率 | >= 80% | 10% |
