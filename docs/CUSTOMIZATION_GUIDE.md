# Constitution 客製化指南

> 適用於團隊根據自身需求調整 methodology-v2

---

## 為什麼需要客製化？

```
預設值適合大多數團隊
但每個團隊有不同的：
- 品質標準
- 審批流程
- 命名規範
```

## 1. 調整品質閾值

### 預設值

| 維度 | 閾值 | 權重 |
|------|------|------|
| 正確性 | >= 80% | 30% |
| 安全性 | >= 100% | 25% |
| 可維護性 | >= 70% | 20% |
| 效能 | >= 80% | 15% |
| 覆蓋率 | >= 80% | 10% |

### 建議

| 團隊類型 | 建議閾值 |
|----------|----------|
| 新創團隊 | correctness: 70, coverage: 60 |
| 企業團隊 | correctness: 90, coverage: 90 |
