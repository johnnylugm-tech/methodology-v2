## 使用方式

### 自動生成

```bash
python -m methodology-v2.requirement_traceability --project-id my-project --verify
```

### 匯出報告

```bash
python -m methodology-v2.requirement_traceability --project-id my-project --export traceability_report.json --format aspice
```
