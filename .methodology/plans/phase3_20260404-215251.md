## 8. Iteration 修復流程

```
[FR-{FR_NUM}]
 ↓
[Developer spawn] → confidence < 6？（yes → 重新派遣，最多 3 次）
 ↓
[本地驗證] pytest + coverage → FAIL？（yes → 修復 → re-verify）
 ↓
[Reviewer spawn] → APPROVE？（no → 修復 → re-spawn）
 ↓
[commit] → [sessions_spawn.log 補記 commit hash]
 ↓
[next FR]
```
