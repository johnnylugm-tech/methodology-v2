## 📋 預設參數建議

### 語音合成參數

| 參數 | 推薦預設值 | 原因 | 參考來源 |
|------|------------|------|----------|
| **voice** | `zh-TW-HsiaoHsiaoNeural` | 中文台灣標準音色，清晰自然 | 微軟官方 |
| **rate** | `-2%` | 讓 TTS 有停頓感，更接近自然說話 | constitution 版驗證 |
| **volume** | `+0%` | 預設即可，需要時可調高 | 微軟預設 |
| **chunk_size** | `500` | 平衡延遲和質量，適合簡報場景 | 測試經驗 |

### 停頓控制參數

| 標點 | 停頓時間 | 原因 |
|------|----------|------|
| 逗號 `，` | 200ms | 短暫停頓，保持語流 |
| 句號 `。` | 500ms | 段落結束，明顯停頓 |
| 問號 `？` | 500ms | 語句結束，與句號相同 |
| 感嘆號 `！` | 500ms | 強調語氣，需要停頓 |
| 分號 `；` | 300ms | 中間停頓，長於逗號 |
| 換行 `\n` | 1000ms | 段落級停頓，最長停頓 |

## 🔄 參數調整原則

### 場景調優

| 場景 | 建議調整 |
|------|----------|
| **簡報配音** | rate: -2%（停頓感） |
| **有聲書** | chunk_size: 1000（減少分割） |
| **詩詞朗誦** | rate: -5%（更慢更有感情） |
| **機器人語音** | rate: +0%（快速清晰） |

### 調整幅度

- **rate**：建議範圍 `-50%` ~ `+100%`
- **volume**：建議範圍 `-50%` ~ `+50%`
- **chunk_size**：建議範圍 `100` ~ `1500`

---

## 📊 實驗數據

### rate 參數測試

| rate 值 | 聽感評價 | 適用場景 |
|---------|----------|----------|
| +0% | 標準語速，略顯機械 | 機器人語音 |
| -2% | 有停頓感，自然 | **簡報配音（推薦）** |
| -5% | 明顯緩慢，有感情 | 有聲書、詩詞 |
| +10% | 快速，略顯急促 | 緊急通知 |

### chunk_size 測試

| 大小 | 優點 | 缺點 |
|------|------|------|
| 300 | 分割細，延遲低 | 片段太多 |
| **500** | **平衡點** | **推薦** |
| 800 | 片段適中 | 單一檔案較大 |
| 1000 | 片段少 | 延遲較高 |

---

## ⚠️ 常見錯誤

1. **使用 +0% 作為預設**：會顯得機械，建議用 -2%
2. **chunk_size 過大**：超過 1000 字可能導致 TTS API 超時
3. **忽略換行符**：換行應該觸發更長停頓（1000ms）
