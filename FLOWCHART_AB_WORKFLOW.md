                              【強制檢查清單】
══════════════════════════════════════════════════════════════════════════════

```
每次 Commit 前必須勾選：

  ☐ HybridWorkflow mode = ON（不是 OFF/HYBRID）
  ☐ team 有 2+ Agent（不是只有 1 個）
  ☐ Developer 寫完 code
  ☐ Reviewer 審查並 APPROVE（不是 Developer）
  ☐ Tester 執行 pytest 實際通過
  ☐ 交換角色後，第二個 Reviewer 也 APPROVE
  ☐ Quality Gate 分數 >= 90
  ☐ commit message 有兩個人確認

如果沒做到？→ Commit 會被阻止！
```
